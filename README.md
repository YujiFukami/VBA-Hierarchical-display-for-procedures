# VBA-Hierarchical-display-for-procedures
VBA階層化表示フォーム

###使い方

①「frmKaiso.fm」と「ModKaiso.bas」をダウンロードし、VBEにインポートする。

②セルに「=Kaiso()」と入力するとプロシージャの一覧、階層化表示のフォームが出現する。
